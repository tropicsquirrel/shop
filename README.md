# DEFCON Badge Shop

A landing page for selling handmade DEFCON electronic badges, designed and assembled by a 12-year-old hacker.

## Pages

- `index.html` — Main landing page
- `styles.css` — Stylesheet

## Features

- Hero section with inline SVG badge illustration
- Badge specs & features
- Creator story ("made by a kid")
- Pricing / order section
- FAQ
- Fully responsive, dark DEFCON terminal aesthetic

## Running locally

Open `index.html` in any browser — no build step required.
